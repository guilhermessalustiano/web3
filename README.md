Esta é uma aplicação web que utiliza JWT (JSON Web Token).
Uma simples tela de login, pedindo usuário e senha. Um banco de dados MySQL faz a validação do usuário e senha. Quando o login é realizado com sucesso, gera um token que tem uma curta duração, mas que pode ser renovado.
Aplicação feita na aula de Programação Web 3 no curso de Desenvolvimento de Sistemas na ETEC Bento Quirino.
